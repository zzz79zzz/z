Notes
  • Use this calculator to evaluate the contribution of different test results to an overall Post-test probability of a disease or condition based on the Pre-test probability (prevalence) and the likelihood ratios of the various tests employed.
  • Input one or more test results and the associated LR positive and LR negative values associated with that test.
  • When a test result is positive, only the LR positive input is required. If a test result is negative, only the LR negative input is required.
  • This calculator then mutiplies the appropriate likelihood ratios by the Pre-test odds to produce the Post-test odds.
  • The Post-test odds are then converted to an overall Post-test probability.
  • Positive likelihood ratio: (Test sensitivity/(1 - test specificity); negative likelihood ratio: (1 - test sensitivity)/test specificity).
  • LR: likelihood ratio.

 
Equations used
 
Pre-test odds = Pre-test probability / (1 - Pre-test probability)
Post-test odds = Overall_Odds
Post-test probability = Post-test odds / (1 + Post-test odds)
