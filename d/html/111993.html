Notes
  • A urine osmolal gap of less than 150 mosm/kg in a patient with chronic metabolic acidosis suggests that NH4+ excretion is impaired. Because NH4+ excretion is reduced in patients with distal RTA, a low urine osmolal gap is consistent with this diagnosis.
  • When the urine osmolal gap exceeds 400 mosm/kg, it is likely that the urine NH4+ concentration is 200 mEq/L or greater. This would be expected with hyperchloremic metabolic acidosis generated by chronic diarrhea.
  • Urine glucose can be assumed to be 0 mmol/L if glucose is not detected by urine dipstick testing.
  • Urine Na: urine sodium; Urine K: urine potassium; RTA: renal tubular acidosis.
  • * This calculator uses SI units (Système international d'unités) as the default for inputs. Do not use this calculator if you are using conventional units (also known as US units and gravimetric units); please refer instead to the conventional units version of this calculator.

 
Equations used
 
Urine osmolal gap = Measured urine osmolality - ((2 * (Urine Na + Urine K)) + Urine urea nitrogen + Urine glucose)
