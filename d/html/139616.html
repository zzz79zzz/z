Notes
  • If the Pleural fluid protein is >3.0 g/dL [>30 g/L], then an exudate is likely.
  • If the Pleural fluid cholesterol is >55 mg/dL [>1.42 mmol/L], then an exudate is likely.
  • If the Pleural fluid LDH is greater than two-thirds the upper limit of the laboratory’s normal serum LDH, then an exudate is likely.
  • Please refer to UpToDate topic review on evaluation of pleural effusions for more information.
  • LDH : lactate dehydrogenase enzyme activity; PFP : pleural fluid protein; PFL : pleural fluid LDH; SL : serum LDH; ULN : upper limit normal.


Equations used

PFL_to_ULN_SL_ratio = PFL / ULN_SL
