 APRI interpretation

 
APRI ≤0.3: Unlikely cirrhosis or significant fibrosis
APRI >0.3 and ≤0.5: Unlikely cirrhosis, significant fibrosis possible
APRI >0.5 and ≤1.5: Significant fibrosis or cirrhosis possible
APRI >1.5 and ≤2: Likely significant fibrosis, cirrhosis possible
APRI >2: Likely cirrhosis

 
 

 
Notes
  • Platelets is the platelet count.
  • The normal reference range for Platelets is approximately 150 to 400 X 109/L (SI units) [150000 to 400000/microL (conventional units)].
  • * This calculator uses SI units (Système international d'unités) as the default for inputs. Do not use this calculator if you are using conventional units (also known as US units and gravimetric units); please refer instead to the conventional units version of this calculator.

 
Equations used
 
APRI = 100 * ((AST/Top normal AST)/Platelets)
