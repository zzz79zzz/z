Notes
  • The PELD-Cr score is used to estimate relative disease severity and likely survival of pediatric patients awaiting liver transplantation. PELD is the primary method for ranking priority for liver transplantation in children under age 12.
  • Additional information about the PELD-Cr score is at the OPTN website, https://optn.transplant.hrsa.gov/professionals/by-organ/liver-intestine/.
  • Height and weight Z-scores in this calculator are based on the United States Centers for Disease Control and Prevention (CDC) growth reference.
  • The Z-scores in this calculator are based on the LMS tables from the CDC, which are shown below. The row in the table corresponding to the patient-specific value is found in the first column of the table; values to be used for L (the Box-Cox power), M (the median), and S (the generalized coefficient of variation) in the equations for Z-scores are retrieved from the remaining columns.
  • PELD-Cr: pediatric end-stage liver disease score incorporating creatinine; INR: international normalized ratio; CVVHD: continuous venovenous hemodialysis; OPTN: Organ Procurement & Transplantation Network.

 
Equations used
 
Z-score height = ((Height/M)L - 1) / (L * S)
Z-score weight = ((Weight/M)L - 1) / (L * S)
PELD score = 10 * ((-0.1967 * AgeUsed) - (1.842 * ln(AlbuminUsed)) + BiliFactor + INR_Factor - (0.1807 * MZSUsed) + (1.453 * ln(CreatUsed)) + 1.5287) + 2.82

 

 
Calculation details and variables
  • PELD-Cr scores are rounded to the nearest integer. If the calculated PELD-Cr score is <6, it will be reported as 6.
  • AgeUsed is assigned values by this ordered logic:
    • If Age at most recent laboratory report is <1 year, AgeUsed = 1
    • Else if Age at most recent laboratory report is >5.5 years, AgeUsed = 5.5
    • Else AgeUsed = Age at most recent laboratory report
  • AlbuminUsed is assigned values by this ordered logic:
    • If Albumin is <1 mg/dL, AlbuminUsed = 1 mg/dL
    • Else if Albumin is >1.9 mg/dL, AlbuminUsed = 1.9 mg/dL
    • Else AlbuminUsed = Albumin
  • BiliUsed is assigned values by this ordered logic:
    • If Total bilirubin is <1 mg/dL, BiliUsed = 1 mg/dL
    • If Total bilirubin is >40 mg/dL, BiliUsed = 40 mg/dL
    • Else BiliUsed = Total bilirubin
  • BiliFactor is assigned values by this ordered logic:
    • If BiliUsed is ≤4 mg/dL, BiliFactor = 0.7854 * ln(Bili_Used) + 0.3434 * ln(4)
    • Else BiliFactor = 0.7854 * ln(4) + 0.3434 * ln(BiliUsed)
  • INRUsed is assigned values by this ordered logic:
    • If INR is <1, INRUsed = 1
    • Else if INR is >10, INRUsed = 10
    • Else INRUsed = INR
  • INRFactor is assigned values by this ordered logic:
    • If INRUsed is ≤2, INRFactor = 1.981 * ln(INRUsed) + 0.7298 * ln(2)
    • Else INRFactor = 1.981 * ln(2) + 0.7298 * ln(INRUsed)
  • CreatUsed is assigned values by this ordered logic:
    • If there has been hemodialysis twice or >24 hours of CVVHD in the week prior to the Serum creatinine level, CreatUsed = 1.3 mg/dL
    • Else if Serum creatinine is <0.2 mg/dL, CreatUsed = 0.2
    • Else if Serum creatinine is >1.3 mg/dL, CreatUsed = 1.3
    • Else CreatUsed = Serum creatinine
  • MZSUsed is assigned values by this ordered logic:
    • If the Minimum Z-score (of Height and Weight Z-scores) is <-5, MZSUsed = -5
    • Else if Minimum Z-score is >-2.1, MZSUsed = -2.1
    • Else MZSUsed = Minimum Z-score
