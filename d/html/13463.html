Notes
  • In this calculation, arterial oxygen content CaO2 and venous oxygen content CvO2 are calculated by using both oxygen saturation and oxygen tension of both blood samples.
  • O2 consumption may be determined by:
    • Separate calculation
    • Machine measurement
    • Estimation/assumption of approximately 250 mL/min in a healthy person
  • In this calculator, mcg is the abbreviation for micrograms.
  • Hgb: hemoglobin; O2Sat: oxygen saturation (arterial); PaO2: partial pressure of oxygen (arterial); O2vSat: oxygen saturation (venous); PvO2: partial pressure of oxygen (venous); CaO2: oxygen content (arterial); CvO2: oxygen content (venous).

 
Equations used
 
CaO2 = ( Hgb * 13.4 * O2Sat / 100 ) + ( PaO2 * 0.031 )
CvO2 = ( Hgb * 13.4 * O2vSat / 100 ) + ( PvO2 * 0.031 )
Cardiac output = O2 consumption / (CaO2 - CvO2)
