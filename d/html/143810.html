Notes
  • The MELD 3.0 score accounts for intrinsic differences in Serum creatinine between females and males.
  • In previous iterations of MELD, these differences might bias the calculation and interpretation by resulting in lower MELD scores in females than males with otherwise identical risk factors.
  • If a patient has recently received intravenous albumin, consider using the MELD 3.0 score EXCLUDING serum albumin calculator.
  • For more information on the use of the MELD 3.0 score in organ allocation, please refer to UpToDate topics that discuss the management of liver failure.
  • MELD: Model for End-Stage Liver Disease.

 
Equations used
 
MELD 3.0 score = (Round the following expression to the nearest integer)(Sex + (4.56 * ln(Total bilirubin)) + (0.82 * (137 - Serum sodium)) - (0.24 * (137 - Serum sodium) * ln(Total bilirubin)) + (9.09 * ln(INR)) + (11.14 * ln(Serum creatinine)) + (1.85 * (3.5 - Albumin)) - (1.83 * (3.5 - Albumin) * ln(Serum creatinine)) + 6)
90-day survival estimation = 100 * 0.946e(0.17698*MELD 3.0 score-3.56)
60-day survival estimation = 100 * 0.963e(0.17698*MELD 3.0 score-3.56)
30-day survival estimation = 100 * 0.981e(0.17698*MELD 3.0 score-3.56)

 

 
Calculation details and variables
  • Total bilirubin, Serum creatinine, and INR values are set to 1 if measured values are less than 1.
  • Serum creatinine is set to 3 mg/dL for all measured values >3 mg/dL.
  • Serum sodium is similarly bound 125-137 mEq/L.
  • Serum albumin is similarly bound 1.5-3.5 g/dL.

 
Equation parameters, such as Sex, have 2 or more discrete values that may be used in the calculation. The numbers in the parentheses, eg, (1.33), represent the values that will be used.
