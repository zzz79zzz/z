Equations used

Terms = Functional status + ASA physical status + Sepsis + Emergency case + Type of procedure - 1.7397
Postoperative respiratory failure risk* = 100 * e (Terms) / (1 + e (Terms) )




Equation parameters, such as Functional status , have 2 or more discrete values that may be used in the calculation. The numbers in the parentheses, eg, (0), represent the values that will be used.
