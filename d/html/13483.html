Calculator: Absolute neutrophil count


 Calculator: Absolute neutrophil count

 
Input
 
WBC count   cells/microL
PMNs   %
Bands   %

 
Results
 
Important: Inputs must be complete to perform calculation.

 
 
ANC   cells/microL
 

 
 

 
Notes
  • WBC: white blood cell; PMN: polymorphonuclear cell; ANC: absolute neutrophil count.

 
Equations used
 
ANC = WBC_count * ((PMNs/100) + (Bands/100))

 
Only digits 0 to 9 and a single decimal point (".") are acceptable as numeric inputs. Attempted input of other characters into a numeric field may lead to an incorrect result.
