Notes
  • The MPM0-III predicts outcome in the critically ill population as a whole and does not predict mortality for a single individual.
  • If risk factors, other than age and code status, are all absent, then -0.4243604 is used as the value of Terms 1.
  • The coefficients in parentheses are the logarithms of the odds ratios for each risk factor.

 
Equations used
 
Terms 1 = Coma + HR + SBP + CRI + Cirr + Mets + ARF + Arr + CVA + GIB + IME + CPR + Vent + Med
Terms 2 = Terms 1 + (Age * 0.0385582) + Full - ((Coma > 0) * Age * 0.0075284) - ((SBP > 0) * Age * 0.0085197) - ((Cirr > 0) * Age * 0.022433) - ((Mets > 0) * Age * 0.0330237) - ((Arr > 0) * Age * 0.0101286) - ((IME > 0) * Age * 0.0169215) - ((CPR > 0) * Age * 0.011214) - 5.36283
Mortality_probability = 100 * e(Terms 2) / (1 + e(Terms 2))
