Notes
  • The input “ECG heart rate measure” is either the RR interval in milliseconds of the beat prior to the QT interval, or the heart rate in beats per minute.
  • For more information on the clinical usefulness of the corrected QT interval calculations, please see UpToDate topics on basic electrocardiogram interpretation and both acquired and congenital long QT syndrome.
  • ECG: electrocardiogram; BPM: beats per minute.

 
Equations used
 
QT_Interval_corrected_Bazett = QT_Interval / sqr(RR_interval/1000)
QT_Interval_corrected_Fridericia = QT_Interval / (RR_interval/1000)1/3
QT_Interval_corrected_Framingham = QT_Interval + (0.154 * (1000 - RR_interval))
QT_Interval_corrected_Hodges = QT_Interval + (1.75 * (Heart_rate - 60))
