Mobitz type I (Wenckebach) AV block
Sinus rhythm with Mobitz type I (Wenckebach) atrioventricular block. Note the progressive prolongation of the PR interval with the sixth P wave not being conducted (arrow). This is followed by resumption of AV conduction as the seventh P wave is conducted with a relatively short PR interval that then begins to increase with the next P wave.
