Type A and Type B nuchal cords

Type A and Type B nuchal cords
Type A: The placental end crosses over the umbilical end, entangling the neck in an unlocked pattern. This pattern can become undone with fetal movement.
Type B: The placental end crosses under the umbilical end, entangling the neck in a locked pattern. This pattern cannot undo itself and can form a true knot when it passes caudally over the fetal body.
Modified from: Collins JH. Nuchal cord type A and type B. Am J Obstet Gynecol 1997; 177:94.
