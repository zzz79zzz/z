Formulas to estimate body surface area (BSA)
Boyd*[1] BSA (m2) = 0.0003207 x Ht (cm)0.3 x Wt (g)(0.7285 – ( 0.0188 x LOG(Wt (g))))
DuBois and DuBois*[2] BSA (m2) = Wt (kg)0.425 x Ht (cm)0.725 x 0.007184
Gehan and George*[3] BSA (m2) = Wt (kg)0.51456 x Ht (cm)0.42246 x 0.0235
Haycock*[4] BSA (m2) = Wt (kg)0.5378 x Ht (cm)0.3964 x 0.024265
Mosteller*[5] BSA (m2) = Square root ((Ht (cm) x Wt (kg))/3600)
Yu[6] BSA (cm2)¶ = 71.3989 x Ht (cm)0.7437 x Wt (kg)0.4040
LivingstonΔ[7] BSA (m2) = 0.1173 x Wt (kg)0.6466
Ht: height; SQRT: square root; Wt: weight.
* Recommended by American Society of Clinical Oncology guidelines.
¶ Original formula estimated BSA in cm2; to convert to m2, divide result by 10,000.
Δ Only formula derived from a cohort containing significant numbers of obese persons.
