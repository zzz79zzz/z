Stent with string
The extraction string for the ureteral stent is looped through the distal stent (inset) such that pulling on it will unfurl the loop to make stent removal easier. If the urologist chooses not to leave a string attached to the stent (as in the case when longer indwelling times are required), the string can be cut off and removed.
