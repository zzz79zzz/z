Revised Screener and Opioid Assessment for Patients with Pain (SOAPP-R)
The following are some questions given to patients who are on or being considered for medication for their pain. Please answer each question as honestly as possible. There are no right or wrong answers.
 
  Never Seldom Sometimes Often Very often
0 1 2 3 4
  1. How often do you have mood swings?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you felt a need for higher doses of medication to treat your pain?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you felt impatient with your doctors?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you felt that things are just too overwhelming that you can't handle them?
 [ ] [ ] [ ] [ ] [ ]
  1. How often is there tension in the home?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you counted pain pills to see how many are remaining?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you been concerned that people will judge you for taking pain medication?
 [ ] [ ] [ ] [ ] [ ]
  1. How often do you feel bored?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you taken more pain medication than you were supposed to?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you worried about being left alone?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you felt a craving for medication?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have others expressed concern over your use of medication?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have any of your close friends had a problem with alcohol or drugs?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have others told you that you had a bad temper?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you felt consumed by the need to get pain medication?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you run out of pain medication early?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have others kept you from getting what you deserve?
 [ ] [ ] [ ] [ ] [ ]
  1. How often, in your lifetime, have you had legal problems or been arrested?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you attended an AA or NA meeting?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you been in an argument that was so out of control that someone got hurt?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you been sexually abused?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have others suggested that you have a drug or alcohol problem?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you had to borrow pain medications from your family or friends?
 [ ] [ ] [ ] [ ] [ ]
  1. How often have you been treated for an alcohol or drug problem?
 [ ] [ ] [ ] [ ] [ ]
Please include any additional information you wish about the above answers.
Thank you.
